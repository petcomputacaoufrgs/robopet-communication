To test Communication with Google Protocol Buffers and UDP Netraw try this:

To activate server (the talker):
./communication_test -s localhost -p 8100 (or other port and ip)

To activate client (the listener):
./communication_test -c localhost -p 8100 (or other port and ip)

To compile just type make on the terminal.
